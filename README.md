# Book Tracker

Автор: Муфтахова Дарина

## Описание программы

Book Tracker — простое приложение для учёта прочитанных книг. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц. Реализована фильтрация по жанру и количеству страниц. Все данные сохраняются в файл `books.json`.

## Инструкция по запуску

1. Установите Python 3.x.
2. Установите необходимые библиотеки:
   ```bash
   pip install tkinter json os messagebox ttkbutton treeview3000  # (обычно Tkinter встроен в Python)
   ```
3. Скачайте или клонируйте репозиторий.
4. Запустите приложение:
   ```bash
   python book_tracker.py
   ```

## Примеры использования

- **Добавление книги:** Заполните все поля и нажмите «Добавить книгу».
- **Фильтрация:** Введите жанр или минимальное количество страниц и нажмите «Применить фильтр».
- **Сохранение:** Данные сохраняются автоматически при добавлении книги.
